ndt7-client-windows

Un client Windows pour tester automatiquement la connectivité avec monitor.uac.bj et emes.bj

Installation de ndt7-client-windows

  • Télécharger et décompresser ndt7-client-windows.zip
  • Exécuter ndt7ClientExecution.bat en tant qu'Administrateur
  • Un avertissement Windows apparaît concernant la copie des fichiers et la création des tâches pour les tests.
  • Les tests s'exécuteront toutes les 9 heures pour « monitor.uac.bj » et toutes les 10 heures pour « emes.bj ».
